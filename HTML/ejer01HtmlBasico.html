Cuerpo principal del Contenido

La etiqueta main es en HTML5 una etiqueta semantica.Esto quiere decir que tiene que ver con el significado de lo contenido dentro de ella.Define el contenido principal de la pagina.Unicamente puede haber una sola etiqueta como esta en
todo el documento. El contenido escrito aqui dentro no se repite en el resto del sitio (todo lo contrario a los encabezados de pagina,los menues de navegacion y los pies de pagina).Es usado por buscadores para indexar.
Conviene usar aqui textos que ayuden a encontrar la pagina luego de procesos de indexacion de los buscadores.

Seccion etiquetas semanticas estructurales mas importantes:

  • HEADER
  • FOOTER
  • SECTION
  • ARTICLE
  • NAV
  • ADDRESS

Las etiquetas semanticas estructurales estan relacionadas directamente con el significado del contenido y no con los estilos ni las posiciones dentro del documento.

Reglas para etiquetas semanticas:

  • Los distintos tipos de contenido se puede asociar a las etiquetas SECTION
  • Las unidades de contenido se asocias a las etiquetas ARTICLE o sea que un SECTION anida multiples ARTICLES.
  • Las etiquetas HEADER Y FOOTER pueden ser anidadas dentro HEADER o FOOTER Osea un FOOTER podria anidar otro HEADER y FOOTER.
  • La etiqueta NAV debe ser unica en el documento y corresponde a los elementos de navegacion del documento
  • Reservar etiqueta h1 para usar dentro de etiquetas top level como main,header o footer.

  3. Como se relaciona un header http con la etiqueta head de HTML?

    Los HTTP headers son la parte central de los HTTP requests y responses, y transmiten información acerca del navegador del cliente, de la página solicitada, del servidor, etc.
    Los head provee información general (metadatos) acerca del documento, incluyendo su título y enlaces a scripts y hojas de estilos.

  4. Que es UTF8?

    UTF-8 (8-bit Unicode Transformation Format) es un formato de codificación de caracteres Unicode e ISO 10646 que utiliza símbolos de longitud variable.

  5. Que atributo HTML sirve para aplicar estilo, a que etiqueta es aplicable?

    El atributo style es un atributo que se utiliza en las etiquetas HTML para incrustar código CSS directamente en la propia etiqueta.
    se le aplica el estilo,a un párrafo (p),a una capa con salto de línea (div) o a una capa sin efecto (span), etc.

  6. Donde se indica que un enlace debe ubicar el documento en una nueva pestaña?

  7. Explicar cuando usarían etiquetas semánticas y con que propósito.

    Las etiquetas semánticas ayudan a definir la estructura del documento y permiten que las páginas web sean mejor indexadas por los buscadores.
    Una etiqueta se califica como semántica si tiene que ver el significado, es decir, si nos informa sobre lo que trata su contenido.
    Por ejemplo, la etiqueta SECTION nos dice que contiene una sección o capítulo dentro de la página. Frente a las etiquetas semánticas tenemos
    otros tipos de etiquetas como las que afectan al formato, como UL, a multimedia, como VIDEO, etc.

  8. ¿Que atributos de estilo básicos aplicarían a una tabla? Explicar.

    align: para alinear horizontalmente.
    background: Nos permite colocar un fondo para la tabla a partir de un enlace a una imagen.
    border: Define el número de pixels del borde principal.
    bordercolor: Define el color del borde.
    cellpadding: Define, en pixels, el espacio entre los bordes de la celda y el contenido de la misma.
    cellspacing: Define el espacio entre los bordes (en pixels).
    height: Define la altura de la tabla en pixels o porcentaje.
    width: Define la anchura de la tabla en pixels o porcentaje.

  9. ¿Que diferencia existe entre la etiqueta body de un documento HTML y el body del protocolo HTTP?

    El elemento (body) de HTML representa el contenido de un documento HTML. Solo puede haber un elemento (body) en un documento.
    Un campo de cuerpo de mensaje opcional ('body' en inglés) que lleva los datos asociados con la petición (como contenido de un formulario HTML),
    o los archivos o documentos asociados a una respuesta (como una página HTML, o un archivo de audio, vídeo)

  10. ¿Que atributos marcan la agrupación de celdas de una tabla?. ¿A que etiquetas se aplican?

    Los atributos colspan(columna) y rowspan(fila) permiten unir una celda con las celdas contiguas, tanto horizontal como verticalmente.
    El valor de colspan indica la cantidad de celdas unidas en horizontal y el valor de rowspan indica la cantidad de celdas unidas en vertical.
    A la etiqueta (TD) es la que define la celda de una tabla que contiene datos.
